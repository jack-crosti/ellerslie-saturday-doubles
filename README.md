# Ellerslie Saturday Doubles

A browser-based organiser for Ellerslie Tennis Club's Saturday doubles sessions.

## Live app

[Open Ellerslie Saturday Doubles](https://jack-crosti.github.io/ellerslie-saturday-doubles/)

## Features

- Saved player register with skill ratings from 1 to 5
- Balanced doubles matches across three courts
- Fair waiting-player rotations
- Reduced repeat partnerships
- Adjustable rounds and match duration
- Responsive layout for phones, tablets and computers

Player information is stored locally in the organiser's browser.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
