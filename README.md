# Ellerslie Saturday Doubles

A simple organiser for Ellerslie Tennis Club's Saturday doubles sessions.

## Features

- Select saved players and record skill ratings from 1 to 5
- Create balanced doubles matches across three courts
- Rotate waiting players fairly
- Reduce repeated partnerships
- Choose the number and duration of rounds
- Works on phones, tablets and desktop browsers

Player data is stored locally in each organiser's browser.

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
